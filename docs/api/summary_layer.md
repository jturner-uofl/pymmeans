# `pymmeans.summary_layer`

This module hosts the R-`emmeans`-parity inference layer: `summary`,
`confint`, `test`, `update`, and `as_r_frame`.
discovered that the `docs/api/summary.md` page documented the
bootstrap module (`pymmeans.summary`) but not the R-parity
`pymmeans.summary_layer` module — five of pymmeans's 71 public
symbols were absent from the rendered API reference.
